## Adding New Scripts ğŸ“

To add a new script to the toolbox, update the `scripts` dictionary in `toolbox.py`:

```python
self.scripts: Dict[str, dict] = {
    "your-script-id": {
        "name": "Your Script Name",
        "path": "scripts/your-script.py",
        "description": "What your script does",
        "category": "Script Category",
        "args": ["--arg1", "--arg2"]  # Optional arguments
    }
}
```
